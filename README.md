# vue-cli3

## 本地开发
启动单模块：  
`npm run serve --moudle=index #index模块`  
启动所有模块：  
`npm run serve`  

## 打包  
打包单模块：  
`npm run build --moudle=index`  
<del>打包所有模块：npm run build 暂时没适配打包所有模块的配置</del>

npm install
